## Conventions (all labs)

- `starter/` is yours to edit; `examples/` holds the completed reference.
- `tests/run_tests.sh` (or the lab’s declared test command) must exit 0 before a lab counts as done.
- `expected-output/` contains real captured runs, never fabricated text.
- Nothing needs sudo, accounts, or API keys unless `metadata.yml` says so — and a free alternative is always given.
